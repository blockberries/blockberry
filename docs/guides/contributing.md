# Contributing to Blockberry

Guide for contributing to the Blockberry project.

## Ways to Contribute

- Report bugs
- Fix issues
- Add features
- Improve documentation
- Review pull requests

## Development Workflow

1. Fork repository
2. Create feature branch
3. Make changes
4. Write tests
5. Run linters
6. Submit pull request

## Code Standards

- Follow Effective Go
- Write godoc comments
- Add unit tests
- Run race detector
- Update documentation

## Pull Request Process

1. Open PR with description
2. Pass CI checks
3. Code review
4. Address feedback
5. Merge
