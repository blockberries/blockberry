# Monitoring & Observability Guide

Comprehensive guide to monitoring Blockberry nodes.

## Metrics

Blockberry exposes Prometheus metrics for:

- Block height and latency
- Transaction throughput
- Peer connections
- Mempool size
- Consensus state

## Tracing

OpenTelemetry distributed tracing support.

## Logging

Structured logging with slog.

## Alerting

Set up alerts for:

- Node down
- Falling behind
- High error rate
- Resource exhaustion
